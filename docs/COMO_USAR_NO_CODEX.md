# COMO_USAR_NO_CODEX

## Onde colocar
- Coloque `AGENTS.md` na raiz do projeto.
- Coloque os arquivos da pasta `docs/` dentro do repositório.

## Como iniciar uma sessão no Codex
Use prompts curtos e diretos, por exemplo:

- "Leia o AGENTS.md e os arquivos em docs/. Continue o próximo mega lote sem quebrar o offline."
- "Siga AGENTS.md. Faça o maior lote seguro focando impressão nativa e validação Tauri."
- "Siga AGENTS.md e PROJECT_STATUS.md. Corrija os erros atuais e entregue somente os arquivos alterados."

## O que o agente deve fazer primeiro
1. Ler `AGENTS.md`
2. Ler `docs/CODEX_MASTER_RULES.md`
3. Ler `docs/PROJECT_STATUS.md`
4. Ler `docs/NEXT_MEGA_BATCH.md`
5. Só depois começar a editar
